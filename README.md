# Dissemble

A very WIP disassembler (currently) supporting the 6502, written in TypeScript.

## Compilng and running

Requires `node` and `npm`

Compile:
- `npm install`
- `npm run tsc`

Run:
- `node build/main.js` (or `npm start --` to compile as well)
